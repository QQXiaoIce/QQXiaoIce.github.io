配置选项参考

选项 默认值 说明
id 必填 歌曲ID/歌单ID/专辑ID/搜索关键词
server 必填 音乐平台：netease(网易云), tencent(QQ音乐), kugou(酷狗), xiami(虾米), baidu(百度音乐)
type 必填 类型：song(单曲), playlist(歌单), album(专辑), search(搜索), artist(艺人)

更多高级设置和选项请参考MetingJS官方文档。

注意事项

  1. 确保在发布前测试音乐播放器是否正常工作
  2. 不同音乐平台的ID获取方式可能不同
  3. 某些音乐可能因版权限制无法播放

提示：你可以通过音乐平台的网页版URL中找到音乐/歌单的ID。
